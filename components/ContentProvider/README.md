# Android ContentProvider 相关知识
ContentProvider 用于保存和获取数据，并使其对所有应用程序可见。这是不同应用程序间共享数据的唯一方式，因为 Android 没有提供所有应用共同访问的公共存储区。数据能够存储于文件系统、SQLite 数据库或其他方式。
